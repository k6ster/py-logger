# Python logger
Basic Python console logger.

### Usage:
```python
log_suc("Text", [["field1", "data"], ["field2", "data"]])
log_err("Text", [["field1", "data"], ["field2", "data"]])
log_dbg("Text", [["field1", "data"], ["field2", "data"]])
log_wrn("Text", [["field1", "data"], ["field2", "data"]])

# Would clear the terminal
# clear_terminal()
```
