# Python logger
Basic Python console logger.

### Usage:
```python
log_suc("Text", [["feild1", "data"], ["feild2", "data"]])
log_err("Text", [["feild1", "data"], ["feild2", "data"]])
log_dbg("Text", [["feild1", "data"], ["feild2", "data"]])
```
